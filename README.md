# coffe-sales
Situação de aprendizagem para o MBA EXECUTIVO EM MARKETING DIGITAL, SOCIAL MEDIA.

Este conjunto de dados foi extraído de https://www.kaggle.com/datasets/ahmedabbas757/coffee-sales/data, registra as transações de vendas diárias de uma cafeteria na Cidade do Cabo durante março de 2024.
Inclui os horários das transações, os tipos de pagamento (cartão/dinheiro), os nomes dos produtos de café e a receita por transação.

O conjunto de dados foi desenvolvido para ajudar a explorar os hábitos dos clientes e o desempenho dos negócios — perfeito para análise de séries temporais, visualização de dados ou projetos de análise de dados para iniciantes.
